hexo使用图片

发表于 2017-06-19 | 分类于 hexo | |

hexo框架博客使用本地图片；之前尝试的是放入根目录下，再建立起一个upload的文件夹用于存放博客图片，之后发现图片莫名的显示不出，于是重新寻找方法

首先安装hexo插件

安装成功之后，在你创建一批文章时会同时创建同名文件夹；只需将图片放入同名文件夹中，之后在文章中引用即可

config中配置

打开主题_config.xml配置文件：

将_config.yml文件中的配置项post_asset_folder设为true后，执行命令$ hexo new test，在source/_posts中会生成文章test.md和同名文件夹test。将图片资源放在test中，文章就可以使用相对路径引用图片资源了。
